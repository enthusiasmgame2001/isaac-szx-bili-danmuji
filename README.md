# 三只熊弹幕姬

这是一个能直接在游戏显示b站直播间弹幕的mod，需要前置mod：IsaacSocket

## 连接说明

- 连接直播间：

    - 将直播间号复制到系统剪贴板
    
    -   按[LCtrl + v]黏贴直播间号

- 重置连接：按[LCtrl + z]

- 开关弹幕姬：按[LCtrl + x]
